    json (JavaScrypt object notation), yaml (Yet Another Markup Language),
xml (Extensible Markup Language) - типы данных, которые применяются для сериализации
составных сложных объектов, передачи данных клиент <-> сервер, создания файлов конфигурации.
    xml - самый старый из этих типов данных, давно ставший стандартом передачи данных.
Он отличается гибкостью и возможностью описывать сложный древовидные структуры.
поэтому раньше широко применялся в сложных бизнес-проектах.
    json - более современный формат. Подходит для сериализации составных структур из списков
и словарей. Широко используется вместе с javascript и python.
    yaml - наследник json. Отличается хорошей читабельностью. поэтому широко применяется
при написании файлов конфигурации.
